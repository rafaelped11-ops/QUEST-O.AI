# **App Name**: Concurso AI Prep

## Core Features:

- Question Generation: Generate contest questions using the Deepseek AI model based on user-defined topics and difficulty levels.
- Difficulty Adjustment: Allow users to specify the difficulty level of the generated questions.
- Topic Selection: Enable users to select specific topics or subjects for question generation.
- Question Display: Present the generated questions in a clean, readable format with options for multiple-choice or open-ended answers.

## Style Guidelines:

- Primary color: Calm blue (#72BCD4) to promote focus and trust.
- Background color: Light gray (#F0F4F7) for a clean, distraction-free interface.
- Accent color: Soft orange (#FFAB91) to highlight important elements and calls to action.
- Body and headline font: 'Inter', sans-serif, for a modern and readable text.
- Use simple, clear icons to represent different question categories and tools.
- Subtle animations for loading new questions and providing feedback.
